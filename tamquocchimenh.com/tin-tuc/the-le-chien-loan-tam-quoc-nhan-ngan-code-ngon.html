A. CHIÊU BINH

- Ấn Chiêu Binh sau đó đăng nhập tài khoản Kul các Chúa Công sẽ nhận ngay code Chiêu Binh

a. Thể lệ:

Tham gia báo danh và báo danh đạt mốc 60,000 người nhận code xịn

b. Phần thưởng:

- Code báo danh:

+ Mảnh Tướng Trác Việt Tự Chọn *10

+ Tàn Quyển Vạn Năng *10

+ Binh Thư Trác Việt *20

+ KNB *50

- Code đạt mốc khi chiêu mộ đủ 60.000

+ Thẻ Tướng Quan Vũ *1

+ Mảnh Tuyệt Thế Vạn Năng *10

+ Mảnh Trác Việt Vạn Năng *30

+ KNB *200

Lưu ý:

- Code đạt mốc sẽ nhận ở mốc cuối cùng khi các Chúa Công báo danh đủ 60k người

- Số lượng code có hạn, nếu hết code sự kiện sẽ kết thúc trước thời gian sự kiện

Ấn nút Chiêu Binh để tham gia sự kiện

Nhận code Chiêu Binh

- Các Chúa Công hãy chung tay đạt mốc 60,000 lượt chiêu binh sẽ nhận được code Đạt Mốc

- Phải đạt đủ mốc mới nhận được code, không đủ mốc 60,000 lượt báo danh không nhận được code

Hình ảnh Đạt Mốc

B. VÒNG QUAY
a. Thể lệ: 

- Nhận lượt mỗi ngày tham gia quay tại nút Nhận Lượt: 

+ Mỗi ngày đăng nhập vào trang event sẽ nhận được 5 lượt/ngày

+ Ấn vào nút "Mời Bạn Nhận Lượt" sẽ được cộng thêm 1 lượt quay

+ Mời 1 bạn để nhận thêm 1 lượt, tối đa mời 05 bạn

- Ấn nút "QUAY" để tham gia chiêu mộ những thần tướng, để xây dựng đội hình chiến đấu mạnh mẽ

- Khi Chúa Công đã chiêu mộ được thần tướng nào, thần tướng đó sẽ xuất hiện ở bên dưới 1 trong 4 đội hình

- Khi chiêu mộ đủ 5 thần tướng trong một đội hình, nút Nhận Code sáng lên, Chúa Công ấn vào để nhận code

Lưu ý:

- Người được mời phải đăng nhập vào trang teaser mới được tính và không được mời trùng, ví dụ A mời B, thì C không được mời B

- Quay thành công sẽ trừ lượt

- Không quay sẽ không trừ lượt và lượt quay dư hôm nay cộng dồn sang ngày hôm sau

- Ấn "Nhận Code" số lượng tướng sẽ bị trừ đi tướng ứng với đội hình

- Chúa Công muốn nhận code của đội hình khác, phải tham gia "QUAY" chiêu mộ tướng nhé.

- Số lượng code có hạn, nếu hết code sự kiện sẽ kết thúc trước thời gian sự kiện

b. Phần thưởng:
